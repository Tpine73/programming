키워드 설명

		

			

				
					
						·extends 
  클래스를 상속 받음

·implements 
  인터페이스를 상속 받아 모든 메서드를 구현

·ActionListener 
  버튼 클릭이나 타이머 활성화 등의 이벤트 발생시 actionPerformed 메서드를 호출하는 리스너

·addActionListener 
  ActionListner를 추가

·actionPerformed 
  버튼 클릭이나 타이머 활성화 등의 이벤트 발생시 호출되는 메서드
