5) 키워드 설명
			
·windowClosing 
  JFrame에 있는 종료(X) 버튼을 클릭하면 호출되어 실행하는 메서드

·Runnable 
  스레드에서 run 메서드를 호출할 수 있도록 지원하는 인터페이스

·run 
  스레드의 start 메서드에서 Runnable 객체의 run 메서드를 호출하며, run 메서드에서 실질적으로 스레드 작업을 수행 
  public void run() Runnable 인터페이스를 상속받은 객체에 메서드 정의

·ServerSocket 
  서버에서 네트워크를 개설하기 위한 소켓

·Socket 
  특정 IP주소와 포트번호의 서버에 클라이언트가 연결 요청하면 서버는 수락하고 
 서버-클라이언트를 1:1로 연결하기 위한 네트워크 소켓

·Thread 
  메인 스레드와 별개로 작동하며, 네트워크에 상대편의 데이터가 도착할 때까지 계속 대기하고 있다가 데이터가 도착하면 활성화되는 스레드 
  public class Thread extends Object implements Runnable 
  생성자는 Thread(), Thread( Runnable 객체 ) 등이 가능

·accept 
  클라이언트에서 서버로 연결 요청시 이를 받아들여 연결 활성화  
  public Socket accept()에서 반환값은 클라이언트와 1:1로 연결된 소켓

·start 
  스레드를 시작하는 메서드

·interrupt 
  스레드를 중단하는 메서드

·WindowListener 
  윈도우 창에 이벤트가 발생하면 windowClosing 등의 관련 메서드를 호출할 수 있도록 지원하는 인터페이스
