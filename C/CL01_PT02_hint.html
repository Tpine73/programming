박코치의 C 프로그래밍


문제

						하트 모양을 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 실행순서 및 메모리 상태는 키보드의 A(이전)및 D(다음)를 눌러 확인하세요

입출력 예제

.d88b.d88b.
88888888888
`Y8888888Y'
  `Y888Y'
    `Y'

88888888888
`Y8888888Y'
  `Y888Y'
    `Y'


.d88b.d88b.
88888888888
`Y8888888Y'
  `Y888Y'
    `Y'

알고리즘

						프로그램시작

   하트 모양을 출력

프로그램 종료

프로그램 코드

	#include <stdio.h> 					 

	 

	// 프로그램시작 

1	int main() 						 

	{ 

		// 하트 모양을 출력 

2		printf( "\033[0;31m .d88b.d88b. \033[0m \n" ); 	 

3		printf( "\033[0;33m 88888888888 \033[0m \n" );  	 

4		printf( "\033[0;32m `Y8888888Y' \033[0m \n" );  	 

5		printf( "\033[0;36m  `Y888Y' \033[0m \n" ); 	 

6		printf( "\033[0;34m    `Y' \033[0m \n" );  	 

	 

		// 프로그램 종료 

7		return 0; 

	} 

		
		

	
	
	

		
키워드 설명

		

			

				
					
						·include 
  헤더파일(.h)을 소스코드(.c)에 포함

·stdio.h 
  standard input output의 줄임말로, printf나 scanf와 같이 표준입출력과 관련된 함수 정보를 가지고 있는 헤더파일

·main 
  프로그램을 시작할 때 운영체제가 호출하는 함수

·printf 
  print formatted의 줄임말로, 전달받은 내용을 형식에 맞게 모니터에 출력하는 함수

·\033[0;31m 
  ANSI 색상 코드로 글자색 빨강을 의미

·\033[0m 
  ANSI 색상 코드로 모든 색과 스타일 초기화

·\n 
  줄바꿈(new line) 문자

·\033[0;33m 
  ANSI 색상 코드로 글자색 노랑을 의미

·\033[0;32m 
  ANSI 색상 코드로 글자색 초록을 의미

·\033[0;36m 
  ANSI 색상 코드로 글자색 청록을 의미

·\033[0;34m 
  ANSI 색상 코드로 글자색 파랑을 의미

·return 
  main 함수가 종료하면서 프로그램도 종료
