박코치의 C 프로그래밍


문제

총 프레임 길이 입력 받아서 공이 굴러가는 애니메이션을 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요

입출력 예제

애니메이션 총 프레임 길이를 입력하세요: 5
O

애니메이션 총 프레임 길이를 입력하세요: 10

O

애니메이션 총 프레임 길이를 입력하세요: 20

O

 

		
알고리즘

		
프로그램 시작

   애니메이션 총 프레임 길이 입력 받기

   첫번째 프레임부터 마지막 프레임까지

      0.2초마다 화면 초기화

      프레임마다 이동거리 1칸씩 증가후

      굴러가는 공 출력

프로그램 종료

 
		

	
		
프로그램 코드 
 
	

	
	 
		
	 

	// 파일명 : ./Chapter04/RollingBall.c 

	#include <stdio.h> 

	#include <windows.h>   

	 

	// 프로그램 시작 



1	int main()  

	{ 



2		int frameLength = 0, frame = 0, blank = 0; 

	 

		// 애니메이션 총 프레임 길이 입력 받기 



3		printf( "애니메이션 총 프레임 길이를 입력하세요: "); 

		// 3을 입력한다고 가정 



4		scanf( "%d", &frameLength ); 

	 

		// 첫번째 프레임부터 마지막 프레임까지 



5		for( frame = 1; 				 



6		        frame <= frameLength; 



7		        			frame ++ ) 

		{	 

			// 0.2초마다 화면 초기화  



8			Sleep( 200 );				 



9			system( "cls" );			 

	 

			// 프레임마다 이동거리 1칸씩 증가후  



10			for( blank = 1; 					 



11			        blank <= frame; 



12			        			blank ++ )  

			{ 



13				printf ( " " ); 

			} 

				 

			// 굴러가는 공 출력  



14			printf ( "O" ); 

		} 

	 

		//프로그램 종료  



15		return 0; 

	} 


		
	
		

			

				
				

				  
				

			

			
		
		

	
	

	

		
키워드 설명

		

windows.h 
  system, Sleep 등 윈도우 운영체제 관련 함수 정보를 가지고 있는 헤더 파일


for 
  초기식, 조건식, 증감식으로 구성된 반복문으로 초기식 → 조건식(참) → 명령문 → 증감식 → 조건식(참) → 명령문 → ...  → 증감식 → 조건식(거짓) → for문 종료의 순서대로 실행
for( 초기식; 조건식; 증감식 )
{ 
   조건식이 참이면 실행하는 명령문
}


Sleep 
  일정 시간동안 프로그램을 지연시키는 함수


system 
  명령어를 운영체제에 전달하여 실행시키는 함수


cls 
  clear the screen의 약어로 콘솔 화면을 깨끗하게 초기화하는 운영체제 명령어
