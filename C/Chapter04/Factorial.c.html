박코치의 C 프로그래밍


문제

정수를 입력받아 1부터 입력값까지의 곱 !(factorial)을 계산하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요

입출력 예제

숫자를 입력하세요: 3
3! = 3 * 2 * 1 = 6


숫자를 입력하세요: 5
5! = 5 * 4 * 3 * 2 * 1 = 120


숫자를 입력하세요: 1
1! = 1 = 1

알고리즘

프로그램 시작

   숫자 n 입력받기

   n부터 1까지의 곱 계산하기

   곱셈 결과 출력하기

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter04/Factorial.c 

	#include <stdio.h>  

	 

	// 프로그램 시작 

1	int main()  

	{ 

2		int num = 0, factorial = 1; 

	 

		// 숫자 n 입력받기 

3		printf( "숫자를 입력하세요: "); 

		// 3을 입력한다고 가정 

4		scanf( "%d", &num ); 

		 

		// n부터 1까지의 곱 계산하기 

5		printf( "%d! = ", num ); 

6		for( ; num > 1; 			 

7				num  --  )		 

		{ 

8			printf ( "%d * ", num );	 

9			factorial *= num;			 

		} 

		 

		// 곱셈 결과 출력하기 

10		printf ( "%d = %d", num, factorial );  

	 

		//프로그램 종료 

11		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

for 
  초기식, 조건식, 증감식으로 구성된 반복문으로 초기식 → 조건식(참) → 명령문 → 증감식 → 조건식(참) → 명령문 → ...  → 증감식 → 조건식(거짓) → for문 종료의 순서대로 실행
for( 초기식; 조건식; 증감식 )
{ 
   조건식이 참이면 실행하는 명령문
}


> 
  비교연산자로 ( 왼쪽값 > 오른쪽값 )는 왼쪽값이 오른쪽값보다 크면 참(1)을 반환하고 아니면 거짓(0)을 반환


-- 
  감소연산자로 a--는 a = a - 1를 의미하며 변수공간a  a값 을 변수공간a  a값 - 1 로 변경


*= 
  복합대입연산자로 a *= b는 a = a * b를 의미하며 변수공간a  a값 을 변수공간a  a값 * b값 으로 변경
