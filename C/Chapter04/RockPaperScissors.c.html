문제

사용자가 가위(1), 바위(2), 보(3) 중 하나를 숫자로 선택하지 않으면 다시 안내하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요

입출력 예제

가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 4
가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 2
2를 선택했습니다.


가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 1
1를 선택했습니다.


가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 10
가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 8
가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 6
가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 4
가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 3
3를 선택했습니다.

알고리즘

프로그램시작

   사용자가 가위바위보중 하나를 선택

      사용자가 가위바위보를 잘못 입력하면 다시 선택

   사용자가 선택한 결과를 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter04/RockPaperScissors.c 

	#include <stdio.h>  

 

	// 프로그램시작 

1	int main() 

	{ 

2		int player = 0; 

	 

3		do {	  

			// 사용자가 가위바위보중 하나를 선택 

4			printf( "가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: " ); 

			// 첫 번째는 4, 두 번째는 2를 입력한다고 가정 

5			scanf( "%d", &player ); 

	 

		// 사용자가 가위바위보를 잘못 입력하면 다시 선택 

6		} while ( ( player < 1 ) || ( 3 < player ) );	  

		 

		// 사용자가 선택한 결과를 출력 

7		printf( "\n%d를 선택했습니다.\n", player ); 

		 

		// 프로그램 종료 

8		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

do 
  do-while문은 해당 명령문을 먼저 실행하고 조건식을 나중에 확인하는 반복문으로 명령문 → 조건식(참) → 명령문 → ... → 조건식(거짓) → do-while문 종료의 순서대로 실행
do { 
   조건식이 참이면 실행하는 명령문
} while( 조건식 );


while 
  조건식이 참이면 해당 명령문을 실행하는 반복문으로 조건식(참) → 명령문 → 조건식(참) → 명령문 → ... → 조건식(거짓) → while문 종료의 순서대로 실행
while( 조건식 )
{ 
   조건식이 참이면 실행하는 명령문
}


< 
  비교연산자로 ( 왼쪽값 < 오른쪽값 )은 왼쪽값이 오른쪽값보다 작으면 참(1)을 반환하고 아니면 거짓(0)을 반환


|| 
  논리연산자 OR로 ( 왼쪽조건식 || 오른쪽조건식 )은 두 조건식 중 하나라도 참이면 참(1)을 반환하고, 모두 거짓이면 거짓(0)을 반환
