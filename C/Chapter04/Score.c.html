박코치의 C 프로그래밍


문제

사용자가 0점 ~ 100점 사이의 점수를 입력하지 않으면 다시 안내하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요

입출력 예제

0점 ~ 100점 사이의 점수를 입력하세요: 150
0점 ~ 100점 사이의 점수를 입력하세요: 89
당신의 점수는 89점입니다.


0점 ~ 100점 사이의 점수를 입력하세요: 200
0점 ~ 100점 사이의 점수를 입력하세요: -32
0점 ~ 100점 사이의 점수를 입력하세요: 74
당신의 점수는 74점입니다.


0점 ~ 100점 사이의 점수를 입력하세요: 95
당신의 점수는 95점입니다.

알고리즘

프로그램시작

   점수를 입력

      사용자가 점수를 잘못 입력하면 다시 입력 안내

   사용자가 입력한 점수를 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter04/Score.c 

	#include <stdio.h>  

 

	// 프로그램시작 

1	int main() 

	{ 

		// 점수 초기화 

2		int score = -1; 

	 

3		while( score < 0 )  

		{ 

			// 점수를 입력 

4			printf( "0점 ~ 100점 사이의 점수를 입력하세요: " ); 

			// 첫 번째는 150, 두 번째는 89를 입력한다고 가정 

5			scanf( "%d", &score ); 

	 

			// 사용자가 점수를 잘못 입력하면 다시 입력 안내 

6			if ( ( score < 0 ) || ( 100 < score ) )   

			{ 

7				score = -1; 

8				continue;  

			} 

			 

			// 사용자가 입력한 점수를 출력 

9			printf( "\n당신의 점수는 %d점입니다.\n", score ); 

		} 					 

		 

		// 프로그램 종료 

10		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

while 
  조건식이 참이면 해당 명령문을 실행하는 반복문으로 조건식(참) → 명령문 → 조건식(참) → 명령문 → ... → 조건식(거짓) → while문 종료의 순서대로 실행
while( 조건식 )
{ 
   조건식이 참이면 실행하는 명령문
}


< 
  비교연산자로 ( 왼쪽값 < 오른쪽값 )은 왼쪽값이 오른쪽값보다 작으면 참(1)을 반환하고 아니면 거짓(0)을 반환


|| 
  논리연산자 OR로 ( 왼쪽조건식 || 오른쪽조건식 )은 두 조건식 중 하나라도 참이면 참(1)을 반환하고, 모두 거짓이면 거짓(0)을 반환


continue 
  이후의 명령문은 실행하지 않고 반복문의 조건식으로 실행흐름을 바꾸는 명령문으로 조건식(참) → 명령문 → continue → 조건식(참) → ...의 순서대로 실행
while( 조건식 )
{ 
   조건식이 참이면 실행하는 명령문
   ...
   continue;
   continue가 적용되지 않을 때만 실행하는 명령문
}
