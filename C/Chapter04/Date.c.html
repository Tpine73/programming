프로그램 시작

   시작요일 및 말일 입력 받기

   1일 이전은 공백 출력

   1일부터 말일까지 출력

      새로운 일주일은 새 줄에 출력

      일주일을 한줄에 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter04/Date.c 

	#include <stdio.h>  

	 

	// 프로그램 시작 

1	int main(void)  

	{ 

		// 날짜 및 요일 초기화 

		int day = 0; 

		int lastDay = 0; 

		int week = 0; 

		int sunday = 0;		 

2		int firstDay = 0; 

	 

		// 시작요일 및 말일 입력 받기 

		// 시작요일을 입력 받기 

3		printf( "일(0),월(1),화(2),수(3),목(4),금(5),토(6) 중에서 \n이번달 1일은 무슨요일인가요 : " ); 

		// 화요일(2)을 입력한다고 가정 

4		scanf("%d", &firstDay ); 

		 

		// 말일 입력 받기  

5		printf( "이번달 말일은 언제인가요 : " ); 

		// 9일을 입력한다고 가정 

6		scanf("%d", &lastDay ); 

	 

		// 1일 이전은 공백 출력 

7		for( week = sunday; 										 

8					week < firstDay;  

9									week ++  )				 

		{ 

10			printf( "     " ); 

		} 

		 

		// 1일부터 말일까지 출력 

11		for ( day = 1; 									 

12				day <= lastDay;  						 

13							week ++ , day ++  )		 

		{ 

			// 새로운 일주일은 새 줄에 출력

			{ 

15				printf( "\n"); 

			} 

			// 일주일을 한줄에 출력 

16			printf( "%5d", day );									 

		}

키워드 설명

		

for 
  초기식, 조건식, 증감식으로 구성된 반복문으로 초기식 → 조건식(참) → 명령문 → 증감식 → 조건식(참) → 명령문 → ...  → 증감식 → 조건식(거짓) → for문 종료의 순서대로 실행
for( 초기식; 조건식; 증감식 )
{ 
   조건식이 참이면 실행하는 명령문
}


++ 
  증가연산자로 a++은 a = a + 1를 의미하며 변수공간a  a값 을 변수공간a  a값 + 1 로 변경


<= 
  비교연산자로 ( 왼쪽값 <= 오른쪽값 )은 왼쪽값이 오른쪽값보다 작거나 같으면 참(1)을 반환하고 아니면 거짓(0)을 반환


%5d 
  5칸을 확보하고 정수 형식으로 출력
예를 들어 28은 5칸중에서 3칸에 빈칸을 출력 후 남은 2칸에 28을 출력
