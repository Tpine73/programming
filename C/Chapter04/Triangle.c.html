알고리즘

프로그램 시작

   사용자에게 삼각형의 크기를 입력 받기

   삼각형 크기만큼 한 줄씩 출력

      n번째 줄은 '삼각형 크기 - 줄번호'만큼 공백 ' '을 출력

      n번째 줄은 '2*줄번호 - 1'만큼 별모양 '*'를 출력

      끝에 빈줄을 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter04/Triangle.c 

	#include <stdio.h> 

	 

	// 프로그램 시작 

1	int main(void) 

	{ 

		// 변수 초기화 

2		int size = 0, line = 0, blank = 0, star = 0; 

 

		// 사용자에게 삼각형의 크기를 입력 받기 

3		printf("삼각형 크기를 입력하세요 : "); 

		// 3을 입력받는다고 가정 

4		scanf("%d", &size ); 

	 

		// 삼각형 크기만큼 한 줄씩 출력 

5		for ( line = 1; 

6				line <= size;						 

7					      line  ++  )					 

		{ 

			// n번째 줄은 '삼각형 크기 - 줄번호'만큼 공백 ' '을 출력 

8			for ( blank = size - line;					 

9					      blank > 0; 

10						         blank  --  )		 

			{ 

11				printf(" "); 

			} 

			// n번째 줄은 '2*줄번호 - 1'만큼 별모양 '*'를 출력 

12			for ( star = 2*line-1;					 

13					      star > 0; 

14							star  --  )		 

			{ 

15				printf("*"); 

			} 

	 

			// 끝에 빈줄을 출력 

16			printf("\n"); 

		} 

	 

		// 프로그램 종료 

17		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

for 
  초기식, 조건식, 증감식으로 구성된 반복문으로 초기식 → 조건식(참) → 명령문 → 증감식 → 조건식(참) → 명령문 → ...  → 증감식 → 조건식(거짓) → for문 종료의 순서대로 실행
for( 초기식; 조건식; 증감식 )
{ 
   조건식이 참이면 실행하는 명령문
}


<= 
  비교연산자로 ( 왼쪽값 <= 오른쪽값 )은 왼쪽값이 오른쪽값보다 작거나 같으면 참(1)을 반환하고 아니면 거짓(0)을 반환


++ 
  증가연산자로 a++은 a = a + 1를 의미하며 변수공간a  a값 을 변수공간a  a값 + 1 로 변경


> 
  비교연산자로 ( 왼쪽값 > 오른쪽값 )는 왼쪽값이 오른쪽값보다 크면 참(1)을 반환하고 아니면 거짓(0)을 반환


-- 
  감소연산자로 a--는 a = a - 1를 의미하며 변수공간a  a값 을 변수공간a  a값 - 1 로 변경
