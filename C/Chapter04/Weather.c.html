문제

온도를 입력받아 섭씨 -100도부터 100도 사이의 온도만 출력하고 종료하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요

입출력 예제

섭씨 -100도부터 100도 사이에서 현재 온도는 몇 도인가요 : 203
섭씨 -100도부터 100도 사이에서 현재 온도는 몇 도인가요 : 4
현재 기온은 섭씨 4도입니다.

섭씨 -100도부터 100도 사이에서 현재 온도는 몇 도인가요 : -302
섭씨 -100도부터 100도 사이에서 현재 온도는 몇 도인가요 : 24
현재 기온은 섭씨 24도입니다.


섭씨 -100도부터 100도 사이에서 현재 온도는 몇 도인가요 : 15

알고리즘

프로그램시작

   온도를 입력

   사용자가 올바르게 입력한 경우만 온도를 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter04/Weather.c 

	#include <stdio.h>  

 

	// 프로그램시작 

1	int main() 

	{ 

		// 온도 초기화 

2		int temperature = 0; 

	 

3		while ( 1 )  

		{  

			// 온도를 입력 

4			printf( "섭씨 -100도부터 100도 사이에서 현재 온도는 몇 도인가요 : " ); 

			// 첫 번째는 203, 두 번째는 4를 입력한다고 가정 

5			scanf( "%d", &temperature ); 

 

			// 사용자가 올바르게 입력한 경우만 온도를 출력 

6			if ( ( -100 <= temperature ) && ( temperature <= 100 ) )  

			{ 

7				printf( "현재 기온은 섭씨 %d도입니다.\n", temperature ); 

8				break;  

			} 

		} 

	 

		// 프로그램 종료 

9		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

while 
  조건식이 참이면 해당 명령문을 실행하는 반복문으로 조건식(참) → 명령문 → 조건식(참) → 명령문 → ... → 조건식(거짓) → while문 종료의 순서대로 실행
while( 조건식 )
{ 
   조건식이 참이면 실행하는 명령문
}


1 
  조건식이 참이라는 의미의 대표적인 정수값


if 
  조건식이 참이면 해당 명령문을 실행하는 선택문
if ( 조건식 ) 
{ 
   조건식이 참이면 실행하는 명령문 
} 


<= 
  비교연산자로 ( 왼쪽값 <= 오른쪽값 )은 왼쪽값이 오른쪽값보다 작거나 같으면 참(1)을 반환하고 아니면 거짓(0)을 반환


break 
  실행흐름을 중단하고 해당 영역을 빠져나오는 명령문으로, while 문이나 for 문에서 break를 실행하면 이후 이후 명령문이나 조건식은 모두 건너띄고 while 문이나 for 문을 빠져나옴 
while( 조건식 )
{ 
   조건식이 참이면 실행하는 명령문
   break; 
   break문이 적용되지 않을 때만 실행하는 명령문
}
