박코치의 C 프로그래밍


문제

						돈을 입력 받아, 500원, 100원, 50원, 10원의 개수를 계산하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 실행순서 및 메모리 상태는 키보드의 A(이전)및 D(다음)를 눌러 확인하세요

입출력 예제

돈을 입력하세요: 3210
3210원을 
500원짜리 동전 6개, 
100원짜리 동전 2개, 
50원짜리 동전 0개, 
10원짜리 동전 1개로 
거슬러 드립니다.


돈을 입력하세요: 1230
1230원을 
500원짜리 동전 2개, 
100원짜리 동전 2개, 
50원짜리 동전 0개, 
10원짜리 동전 3개로 
거슬러 드립니다.


돈을 입력하세요: 100
100원을 
500원짜리 동전 0개, 
100원짜리 동전 1개, 
50원짜리 동전 0개, 
10원짜리 동전 0개로 
거슬러 드립니다.

알고리즘

						프로그램 시작

   사용자에게 돈을 입력받기

   동전 개수를 계산

      잔액에서 500원짜리 동전 개수 계산하고, 잔액 재조정

      잔액에서 100원짜리 동전 개수 계산하고, 잔액 재조정

      잔액에서 50원짜리 동전 개수 계산하고, 잔액 재조정

      잔액에서 10원짜리 동전 개수 계산하고, 잔액 재조정

   동전 개수를 출력

프로그램 종료

프로그램 코드

	#include  

	 

	// 프로그램 시작 

1	int main() 

	{ 

		int payment=0, change=0;								 

2		int coin500=0, coin100=0, coin50=0, coin10=0;					 

	 

		// 사용자에게 돈을 입력받기 

3		printf( "돈을 입력하세요: " ); 

4		scanf( "%d", &payment ); 								 

5		change = payment;									 

 

		// 동전 개수를 계산 

		// 잔액에서 500원짜리 동전 개수 계산하고, 잔액 재조정 

6		coin500 = change / 500;								 

7		change = change % 500;								 

	 

		// 잔액에서 100원짜리 동전 개수 계산하고, 잔액 재조정 

8		coin100 = change / 100;								 

9		change = change % 100;								 

	 

		// 잔액에서 50원짜리 동전 개수 계산하고, 잔액 재조정 

10		coin50  = change / 50;									 

11		change = change % 50;									 

	 

		// 잔액에서 10원짜리 동전 개수 계산하고, 잔액 재조정 

12		coin10  = change / 10;									 

13		change = change % 10;									 

	 

		// 동전 개수를 출력 

14		printf( "%d원을 ", payment ); 

15		printf( "\n500원짜리 동전 %d개, ", coin500 ); 

16		printf( "\n100원짜리 동전 %d개,",  coin100 ); 

17		printf( "\n 50원짜리 동전 %d개,",  coin50 ); 

18		printf( "\n 10원짜리 동전 %d개로", coin10 );  

19		printf( "\n 거슬러 드립니다." ); 

	 

		// 프로그램 종료 

20		return 0; 

	} 

		
		

	
	
	

		
키워드 설명

		

			

				
					
						·int 
  정수 자료형으로 -2, -1, 0, 1, 2 등이 해당

·%d 
  정수형 숫자 형식으로 입출력

·= 
  대입연산자는 a = b에서 변수 공간 a에 b값을 저장하여 a와 b를 같게 함

·/ 
  변수 a, b에 대해 a / b는 a를 b로 나누었을 때의 몫을 의미

·% 
  정수형 변수 a, b에 대해 a % b는 a를 b로 나누었을 때의 나머지를 의미하며, 참고로 몫은 a / b로 연산
