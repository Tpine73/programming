박코치의 C 프로그래밍


문제

8부터 15사이의 10진수를 입력받아 2진수로 변환하여 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요

입출력 예제

8부터 15사이의 10진수를 입력하세요 : 11
10진수 11은 2진수 1011입니다.


8부터 15사이의 10진수를 입력하세요 : 10
10진수 10은 2진수 1010입니다.


8부터 15사이의 10진수를 입력하세요 : 8
10진수 8은 2진수 1000입니다.

알고리즘

프로그램 시작

   8부터 15사이의 10진수를 입력

   10진수에서 2진수의 각 자리수를 추출

      2진수 1000 = 2^3 = 8의 자리수

      2진수 100 = 2^2 = 4의 자리수

      2진수 10 = 2^1 = 2의 자리수

      2진수 1 = 2^0 = 1의 자리수

   2진수 각 자리수의 값을 바탕으로 10진수를 2진수로 변환

   10진수를 2진수로 변환한 결과를 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter02/DecimalToBinary.c 

	#include <stdio.h> 

	 

	// 프로그램 시작 

1	int main() 

	{ 

		// 변수 선언 및 초기화 

		int binary = 0, binary1000 = 0, binary100 = 0, binary10 = 0, binary1 = 0; 

2		int decimal = 0, temp = 0; 

	 

		// 8부터 15사이의 10진수를 입력 

3		printf( "8부터 15사이의 10진수를 입력하세요 : " ); 

4		scanf( "%d", &decimal ); 

5		temp = decimal;  

	 

		// 10진수에서 2진수의 각 자리수를 추출 

		// 2진수 1000 = 2^3 = 8의 자리수 

6		binary1000 = temp / 8;					 

7		temp = temp % 8;					 

	 

		// 2진수 100 = 2^2 = 4의 자리수 

8		binary100 = temp / 4;					 

9		temp = temp % 4;					 

	 

		// 2진수 10 = 2^1 = 2의 자리수 

10		binary10 = temp / 2;					 

11		temp = temp % 2;					 

	 

		// 2진수 1 = 2^0 = 1의 자리수 

12		binary1 = temp; 

	 

		// 2진수 각 자리수의 값을 바탕으로 10진수를 2진수로 변환 

13		binary = binary1000 * 1000 + binary100 * 100 + binary10 * 10 + binary1;	 

	 

		// 10진수를 2진수로 변환한 결과를 출력 

14		printf( "10진수 %d은 2진수 %d입니다.\n", decimal, binary ); 

	 

		// 프로그램 종료 

15		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

/ 
  나눗셈 연산자


8 
  이진수 1000 = 1*23 + 0*22 + 0*21 + 0*20에 해당하는 십진수는 8


% 
  나머지 연산자로 정수 a, b에 대해 a % b는 a를 b로 나누었을 때의 나머지를 연산하며, 참고로 몫은 a / b로 연산
 예를 들어 14 % 4 = 2 이고 14 / 4 = 3


4 
  이진수 100 = 1*22 + 0*21 + 0*20에 해당하는 십진수는 4


2 
  이진수 10 = 1*21 + 0*20에 해당하는 십진수는 2
