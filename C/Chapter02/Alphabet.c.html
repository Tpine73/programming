박코치의 C 프로그래밍


문제

알파벳 소문자를 입력받아 대문자를 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요

입출력 예제

알파벳 소문자 한 글자를 입력하세요: k
소문자 k의 대문자는 K입니다^^


알파벳 소문자 한 글자를 입력하세요: j
소문자 j의 대문자는 J입니다^^


알파벳 소문자 한 글자를 입력하세요: h
소문자 h의 대문자는 H입니다^^

알고리즘

프로그램 시작

   알파벳 소문자를 입력받기

   알파벳 소문자를 대문자로 변환하기

   입력값 출력하기

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter02/Alphabet.c 

	#include <stdio.h>  

 

	// 프로그램 시작 

1	int main()									 

	{ 

		char lowercase = '\0';							 

2		char uppercase = '\0';							 

	 

		// 알파벳 소문자를 입력받기 

3		printf("알파벳 소문자 한 글자를 입력하세요: " ); 

		// k를 입력한다고 가정 

4		scanf( "%c", &lowercase ); 

	 

		// 알파벳 소문자를 대문자로 변환하기 

5		uppercase = lowercase - 'a' + 'A';					 

	 

		// 입력값 출력하기 

6		printf("소문자 %c의 대문자는 %c입니다^^", lowercase, uppercase );	 

	 

		// 프로그램 종료 

7		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

int 
  -2, -1, 0, 1, 2 등의 정수 자료형


char 
  'A', 'b', '3', '!' 등의 1글자 문자 자료형이며, 아스키 코드 테이블을 바탕으로 각 문자는 정수에 매핑


%c 
  1글자의 문자 형식으로 입출력


'a' 
  lowercase 변수 값에서 소문자 특징을 빼기 위한 상수 값 97


'A' 
  lowercase 변수 값에 대문자 특징을 부여하기 위한 상수 값 65
