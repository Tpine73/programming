프로그램시작

   사용자가 체험학습 시작일을 입력

   사용자가 체험학습 종료일을 입력

   체험학습 기간이 몇박 몇일인지 계산

   사용자가 입력한 체험학습 시작일, 종료일, 기간을 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter02/Date.c 

	#include <stdio.h>  

 

	// 프로그램시작 

1	int main() 

	{ 

		// 체험학습 시작일, 종료일 초기화 

		int startDate=0;										 

		int endDate=0;  

		int nights=0;  

2		int days=0;  

	 

		// 사용자가 체험학습 시작일을 입력 

3		printf( "이달 1일부터 31일 중에서 체험학습 시작일을 입력하세요: " ); 

		// 15일을 입력한다고 가정 

4		scanf( "%d", &startDate ); 

	 

		// 사용자가 체험학습 종료일을 입력 

5		printf( "이달 1일부터 31일 중에서 체험학습 종료일을 입력하세요: " ); 

		// 18일을 입력한다고 가정 

6		scanf( "%d", &endDate ); 

	 

		// 체험학습 기간이 몇박 몇일인지 계산 

7		nights = endDate - startDate;							 

8		days = endDate - startDate + 1;							 

	 

		// 사용자가 입력한 체험학습 시작일, 종료일, 기간을 출력 

9		printf( "\n당신은 체험학습을 %d일부터 %d일까지 %d박%d일 다녀오는 군요", startDate, endDate, nights, days ); 

10		printf( "\n뜻 깊고 즐거운 시간 보내시길 바래요 ♪~ ♬~~ \n" );		 

	 

		// 프로그램 종료 

11		return 0; 

	}

  대입 연산자로 a=b는 변수공간a  a값 을 변수공간a  b값 으로 변경


- 
  뺄셈 연산자


+ 
  덧셈 연산자
