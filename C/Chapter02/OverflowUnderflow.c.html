박코치의 C 프로그래밍


문제

변수에 값을 저장할 때 발생하는 overflow와 underflow를 실습하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요

입출력 예제

max     = 2147483647
max + 1 = -2147483648 ( overflow발생 ) 
min     = -2147483648
min - 1 = 2147483647 ( underflow발생 )


max     = 2147483647
max + 1 = -2147483648 ( overflow발생 )

min - 1 = 2147483647 ( underflow발생 )

알고리즘

프로그램 시작

   int에 저장할 수 있는 최대값과 최소값으로 초기화

   int의 최대값에 1을 더하면 overflow 발생

   int의 최소값에 1을 빼면 underflow 발생

   overflow와 underflow를 실습 결과 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter02/OverflowUnderflow.c 

	#include <stdio.h> 

	 

	// 프로그램 시작 

1	int main() 

	{	 

		// int에 저장할 수 있는 최대값과 최소값으로 초기화 

		int max = +2147483647;			 

2		int min = -2147483648;			 

	 

		// int의 최대값에 1을 더하면 overflow 발생 

3		int overflow = max + 1;			 

	 

		// int의 최소값에 1을 빼면 underflow 발생 

4		int underflow = min - 1;	  

	 

		// overflow와 underflow를 실습 결과 출력  

5		printf( "max     =  %d\n", max ); 

6		printf( "max + 1 = %d ( overflow 발생 )\n", overflow );	  

7		printf( "min     = %d\n", min ); 

8		printf( "min - 1 =  %d ( underflow 발생 )\n", underflow );  

	 

		// 프로그램 종료 

9		return 0; 

	}
