박코치의 C 프로그래밍


문제

1.6km까지 기본료가 4800원이고 131m 당 100원이 부과된다고 가정할 때, 1.6km 이상의 이동거리(m)를 입력하면 택시비를 계산하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요

입출력 예제

택시의 이동거리(m)를 입력하세요: 1600
1600m 이동시 택시요금은 4800원입니다


택시의 이동거리(m)를 입력하세요: 1601
1601m 이동시 택시요금은 4900원입니다


택시의 이동거리(m)를 입력하세요: 3600
3600m 이동시 택시요금은 6400원입니다

알고리즘

프로그램 시작

   사용자에게 이동거리를 입력받기

   택시요금 계산

      택시요금은 1.6km까지 기본료가 4800원이고 131m 당 100원 부과

   택시요금 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter02/TaxiFare.c 

	#include <stdio.h> 

	 

	// 프로그램 시작 

1	int main() 

	{ 

2		int distance = 0, taxiFare = 0;								 

	 

		// 사용자에게 이동거리를 입력받기 

3		printf( "택시의 이동거리(m)를 입력하세요: " ); 

		// 1600을 입력한다고 가정 

4		scanf( "%d", &distance ); 								 

		 

		// 택시요금 계산 

		// 택시요금은 1.6km까지 기본료가 4800원이고 131m 당 100원 부과 

5		taxiFare = 4800 + ( distance - 1600 + 130 ) / 131 * 100;				 

	 

		// 택시요금 출력 

6		printf( "%dm 이동시 택시요금은 %d원입니다 ", distance, taxiFare ); 			 

	 

		// 프로그램 종료 

7		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

int 
  -2, -1, 0, 1, 2 등의 정수 자료형


= 
  대입 연산자로 a=b는 변수공간a  a값 을 변수공간a  b값 으로 변경


%d 
  정수 형식으로 입출력


+ 
  덧셈 연산자


- 
  뺄셈 연산자


/ 
  나눗셈 연산자


* 
  곱셈 연산자
