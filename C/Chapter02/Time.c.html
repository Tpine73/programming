박코치의 C 프로그래밍


문제

초를 입력 받아, 시, 분, 초를 계산하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요

입출력 예제

초를 입력하세요 : 3661
3661초는 1시간 1분 1초입니다.


초를 입력하세요 : 61
61초는 0시간 1분 1초입니다.


초를 입력하세요 : 0
0초는 0시간 0분 0초입니다.

알고리즘

프로그램 시작

   사용자에게 전체 초를 입력받기

   시, 분, 초를 계산

      전체 초가 몇 시간인지 계산

      남은 시간이 몇 분인지 계산

      남은 시간이 몇 초인지 계산

   시, 분, 초를 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter02/Time.c 

	#include <stdio.h> 

	 

	// 프로그램 시작 

1	int main() 

	{ 

		int totalSeconds=0, change=0;						 

2		int hours=0, minutes=0, seconds=0;					 

	 

		// 사용자에게 전체 초를 입력받기 

		// 3661초를 입력한다고 가정 

3		printf( "초를 입력하세요 : " ); 

4		scanf( "%d", &totalSeconds ); 							 

5		change = totalSeconds;								 

	 

		// 시, 분, 초를 계산 

		// 전체 초가 몇 시간인지 계산 

6		hours  = change / ( 60 * 60 );							 

	 

		// 남은 시간 재조정 

7		change = change % ( 60 * 60 );							 

	 

		// 남은 시간이 몇 분인지 계산 

8		minutes = change / 60;								 

	 

		// 남은 시간이 몇 초인지 계산 

9		seconds = change % 60;								 

	 

		// 시, 분, 초를 출력 

10		printf( "%d초는 %d시간 %d분 %d초입니다.", totalSeconds, hours, minutes, seconds ); 

	 

		// 프로그램 종료 

11		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

int 
  -2, -1, 0, 1, 2 등의 정수 자료형


%d 
  정수 형식으로 입출력


= 
  대입 연산자로 a=b는 변수공간a  a값 을 변수공간a  b값 으로 변경


/ 
  나눗셈 연산자


% 
  나머지 연산자로 정수 a, b에 대해 a % b는 a를 b로 나누었을 때의 나머지를 연산하며, 참고로 몫은 a / b로 연산
 예를 들어 14 % 4 = 2 이고 14 / 4 = 3
