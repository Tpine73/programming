알고리즘

프로그램 시작

   먼저, 사용자는 동전의 앞면과 뒤면 중 하나를 선택

      사용자가 선택한 동전 상태 출력

   동전을 임의로 던져서 받기

      동전 던져서 받은 결과  출력

   사용자가 동전의 상태를 맞추었는지 결과 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter03/FlipCoin.c 

	#include <stdio.h> 

	#include <stdlib.h>							 

	#include <time.h> 

	 

	// 프로그램 시작 

1	int main() 

	{ 

		int headsOrTails = 0; 

2		int user = 0; 

	 

		// 먼저, 사용자는 동전의 앞면과 뒤면 중 하나를 선택	 

3		printf("동전 던지기 게임을 시작합니다.\n"); 

4		printf("앞면(0), 뒤면(1) 중 하나를 선택하세요 : "); 

		// 0(앞면)을 입력한다고 가정 

5		scanf( "%d", &user ); 

		 

		// 사용자가 선택한 동전 상태 출력		 

6		( user == 0 )	?					 

7			printf( "동전 앞면을 선택했습니다." )  

8		:								 

9			printf( "동전 뒤면을 선택했습니다."); 

 

		// 동전을 임의로 던져서 받기		 

10		srand( time( NULL ) );						 

		// 동전의 상태는 1(뒤면)이 나왔다고 가정 

11		headsOrTails = rand() % 2;					 

	 

		// 동전 던져서 받은 결과  출력		 

12		( headsOrTails == 0 )	?				 

13			printf( "결과는 동전 앞면입니다. " ) 

14		:								 

15			printf("결과는 동전 뒤면입니다. "); 

	 

		// 사용자가 동전의 상태를 맞추었는지 결과 출력	 

16		( headsOrTails == user ) ?				 

17			printf( "맞췄습니다.\n" )  

18		:								 

19			printf("틀렸습니다.\n"); 

	 

		// 프로그램 종료 

20		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

stdlib.h 
  standard library의 줄임말로 rand, malloc 등 자주 사용되는 함수 정보를 가지고 있는 표준 라이브러리 헤더파일


== 
  등가연산자로 ( 왼쪽값 == 오른쪽값 )은 두 값이 같으면 참(1)을 반환하고 다르면 거짓(0)을 반환하며, 대입연산자 =와 비교하여 주의


srand 
  seed for random의 줄임말로 rand 함수에서 매번 다른 패턴으로 임의값을 생성하도록 초기값을 설정하는 함수
초기값이 동일하면 프로그램 새로 시작할 때마다 8467, 15791, 2481와 같은 동일한 패턴으로 임의의 수를 생성하므로 주의 
 rand 함수가 생성한 임의의 수 패턴이 매번 달라지도록 초기값(seed)을 매번 다르게 설정 필요


time 
  시간(초)정보를 가져오는 함수로 time(NULL)은 1970년 1월 1일 0시 0분 0초부터 현재까지의 시간을 초단위로 반환


rand 
  0 ~ 32,767 사이의 정수 중 임의의 수 하나를 반환하는 함수


% 
  나머지 연산자로 정수 a, b에 대해 a % b는 a를 b로 나누었을 때의 나머지를 연산하며, 참고로 몫은 a / b로 연산
 예를 들어 14 % 4 = 2 이고 14 / 4 = 3


else 
  앞에 제시된 조건식이 거짓이면 해당 명령문을 실행하는 선택문 
if ( 조건식 ) 
{ 
   조건식이 참이면 실행하는 명령문 
} 
else 
{ 
   앞에 제시된 조건식이 거짓이면 실행하는 명령문 
}


if 
  조건식이 참이면 해당 명령문을 실행하는 선택문
if ( 조건식 ) 
{ 
   조건식이 참이면 실행하는 명령문 
}
