박코치의 C 프로그래밍


문제

사용자가 입력한 년도가 윤년인지 평년인지 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 참고로, 4로 나누어 떨어지는 해는 윤년이지만, 
그 중에서 100으로 나누어 떨어지는 해는 평년입니다. 
그런데 400으로 나누어 떨어지는 해는 윤년입니다.

입출력 예제

년도를 입력하세요: 2000
2000년은 윤년입니다.


년도를 입력하세요: 2100
2100년은 평년입니다.


년도를 입력하세요: 2020
2020년은 윤년입니다.

알고리즘

프로그램 시작

   사용자가 년도를 입력

   해당년도가 윤년인지 평년인지 출력

      400으로 나누어 떨어지는 해는 윤년으로 출력

      100으로 나누어 떨어지는 해는 평년으로 출력

      4로 나누어 떨어지는 해는 윤년으로 출력

      나머지는 평년으로 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter03/Date.c 

	#include <stdio.h>  

 

	// 프로그램 시작 

1	int main() 

	{ 

		// 년도 초기화 

2		int year = 0; 

 

		// 사용자가 년도를 입력 

3		printf( "년도를 입력하세요: " ); 

		// 2000년을 입력한다고 가정 

4		scanf( "%d", &year ); 

 

		// 해당년도가 윤년인지 평년인지 출력 

		// 400으로 나누어 떨어지는 해는 윤년으로 출력 

5		if ( ( year % 400 ) == 0 )						 

		{ 

6			printf( "%d년은 윤년입니다.", year ); 

		} 

		// 100으로 나누어 떨어지는 해는 평년으로 출력 

7		else if ( ( year % 100 ) == 0 )						 

		{ 

8			printf( "%d년은 평년입니다.", year ); 

		} 

		// 4로 나누어 떨어지는 해는 윤년으로 출력 

9		else if ( ( year % 4 ) == 0 )						 

		{ 

10			printf( "%d년은 윤년입니다.", year ); 

		} 

		// 나머지는 평년으로 출력 

11		else 									 

		{ 

12			printf( "%d년은 평년입니다.", year ); 

		} 

 

		// 프로그램 종료 

13		return 0; 

	}

%

 예를 들어 14 % 4 = 2 이고 14 / 4 = 3
