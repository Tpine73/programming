박코치의 C 프로그래밍


문제

현재 미세먼지 농도를 입력하면 그 결과를 바탕으로 좋음, 보통, 나쁨, 매우나쁨을 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 참고로, 현재 미세먼지 농도( ㎍/m³)가 30이하이면 '좋음', 30 ~ 80이면 '보통', 80~ 150이면 '나쁨', 
151이상이면 '매우나쁨'입니다.

입출력 예제

0 ~ 200 사이에서 현재 미세먼지 농도를 입력하세요 : 60
미세먼지 농도는 대부분의 지역이 '보통' 수준을 보이고 있다


0 ~ 200 사이에서 현재 미세먼지 농도를 입력하세요 : 14
미세먼지 농도는 대부분의 지역이 '좋음' 수준을 보이고 있다


0 ~ 200 사이에서 현재 미세먼지 농도를 입력하세요 : 100
미세먼지 농도는 대부분의 지역이 '나쁨' 수준을 보이고 있다

알고리즘

프로그램 시작

   현재 미세먼지 농도를 입력받기

   현재 미세먼지 농도가 좋음, 보통, 나쁨, 매우나쁨 중 무엇인지를 출력하기

      현재 미세먼지 농도가 30이하이면 좋음 출력

      현재 미세먼지 농도가 30 ~ 80이면 보통 출력

      현재 미세먼지 농도가 80 ~ 150이면 나쁨 출력

      현재 미세먼지 농도가 150보다 높으면 매우나쁨 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter03/FineDust.c 

	#include <stdio.h>  

 

	// 프로그램 시작 

1	int main() 

	{ 

2		int fineDust = 0; 

 

		// 현재 미세먼지 농도를 입력받기  

3		printf("0 ~ 200 사이에서 현재 미세먼지 농도를 입력하세요 : " ); 

		// 60을 입력한다고 가정 

4		scanf( "%d", &fineDust ); 

	 

		// 현재 미세먼지 농도가 좋음, 보통, 나쁨, 매우나쁨 중 무엇인지를 출력하기 

		// 현재 미세먼지 농도가 30이하이면 좋음 출력 

5		if ( fineDust <= 30 )								 

		{ 

6			printf( "미세먼지 농도는 대부분의 지역이 '좋음' 수준을 보이고 있다 " ); 

		} 

		// 현재 미세먼지 농도가 30 ~ 80이면 보통 출력  

7		else if ( ( 30 < fineDust ) && ( fineDust <= 80 ) )				 

		{ 

8			printf( "미세먼지 농도는 대부분의 지역이 '보통' 수준을 보이고 있다 " ); 

		} 

		// 현재 미세먼지 농도가 80 ~ 150이면 나쁨 출력  

9		else if ( ( 80 < fineDust ) && ( fineDust <= 150 ) )				 

		{ 

10			printf( "미세먼지 농도는 대부분의 지역이 '나쁨' 수준을 보이고 있다 " ); 

		} 

		// 현재 미세먼지 농도가 150보다 높으면 매우나쁨 출력  

11		else 										 

		{ 

12			printf( "미세먼지 농도는 대부분의 지역이 '매우나쁨' 수준을 보이고 있다 " ); 

		} 

		 

		// 프로그램 종료 

13		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

if 
  조건식이 참이면 해당 명령문을 실행하는 선택문
if ( 조건식 ) 
{ 
   조건식이 참이면 실행하는 명령문 
} 


<= 
  비교연산자로 ( 왼쪽값 <= 오른쪽값 )은 왼쪽값이 오른쪽값보다 작거나 같으면 참(1)을 반환하고 아니면 거짓(0)을 반환


else if 
  첫번째 조건식이 거짓이고 이후 조건식이 참이면 해당 명령문을 실행하는 선택문
if ( 첫번째 조건식 ) 
{ 
   첫번째 조건식이 참이면 실행하는 명령문 
} 
else if ( 두번째 조건식 ) 
{ 
   두번째 조건식이 참이면 실행하는 명령문 
}
else if ( 세번째 조건식 ) 
{ 
   세번째 조건식이 참이면 실행하는 명령문 
} 
... 
else 
{ 
   앞에 제시된 모든 조건식이 거짓이면 실행하는 명령문 
}


&& 
  논리연산자 AND로 ( 왼쪽조건식 && 오른쪽조건식 )은 두 조건식 모두 참이면 참(1)을 반환하고, 하나라도 거짓이면 거짓(0)을 반환


else 
  앞에 제시된 조건식이 거짓이면 해당 명령문을 실행하는 선택문 
if ( 조건식 ) 
{ 
   조건식이 참이면 실행하는 명령문 
} 
else 
{ 
   앞에 제시된 조건식이 거짓이면 실행하는 명령문 
}
