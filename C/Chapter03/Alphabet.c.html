박코치의 C 프로그래밍


문제

한 글자를 입력받아 그 값이 알파벳 소문자, 알파벳 대문자, 숫자, 기호 중 무엇인지를 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요

입출력 예제

한 글자를 입력하세요: 2
숫자를 입력했군요^^


한 글자를 입력하세요: A
알파벳 대문자를 입력했군요^^


한 글자를 입력하세요: !
기호를 입력했군요^^

알고리즘

프로그램 시작

   글자를 입력받기

   입력값이 알파벳 소문자, 알파벳 대문자, 숫자, 기호 중 무엇인지를 출력하기

      입력값이 알파벳 소문자인 경우 출력하기

      입력값이 알파벳 대문자인 경우 출력하기

      입력값이 숫자인 경우 출력하기

      입력값이 기호인 경우 출력하기

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter03/Alphabet.c 

	#include <stdio.h>  

 

	// 프로그램 시작 

1	int main() 

	{ 

2		char letter = '\0'; 

 

		// 글자를 입력받기  

3		printf("한 글자를 입력하세요: " ); 

		// '2'를 입력한다고 가정 

4		scanf( "%c", &letter ); 

	 

		// 입력값이 알파벳 소문자, 알파벳 대문자, 숫자, 기호 중 무엇인지를 출력하기 

		// 입력값이 알파벳 소문자인 경우 출력하기 

5		if (  ( 'a' <= letter ) && ( letter <= 'z' ) )					 

		{ 

6			printf( "알파벳 소문자를 입력했군요^^" ); 

		} 

		// 입력값이 알파벳 대문자인 경우 출력하기 

7		else if (  ( 'A' <= letter ) && ( letter <= 'Z' ) )				 

		{ 

8			printf( "알파벳 대문자를 입력했군요^^" ); 

		} 

		// 입력값이 숫자인 경우 출력하기 

9		else if (  ( '0' <= letter ) && ( letter <= '9' ) )				 

		{ 

10			printf( "숫자를 입력했군요^^" ); 

		} 

		// 입력값이 기호인 경우 출력하기 

11		else 										 

		{ 

12			printf( "기호를 입력했군요^^" ); 

		} 

		 

		// 프로그램 종료 

13		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

if 
  조건식이 참이면 해당 명령문을 실행하는 선택문
if ( 조건식 ) 
{ 
   조건식이 참이면 실행하는 명령문 
} 


<= 
  비교연산자로 ( 왼쪽값 <= 오른쪽값 )은 왼쪽값이 오른쪽값보다 작거나 같으면 참(1)을 반환하고 아니면 거짓(0)을 반환


&& 
  논리연산자 AND로 ( 왼쪽조건식 && 오른쪽조건식 )은 두 조건식 모두 참이면 참(1)을 반환하고, 하나라도 거짓이면 거짓(0)을 반환


else if 
  첫번째 조건식이 거짓이고 이후 조건식이 참이면 해당 명령문을 실행하는 선택문
if ( 첫번째 조건식 ) 
{ 
   첫번째 조건식이 참이면 실행하는 명령문 
} 
else if ( 두번째 조건식 ) 
{ 
   두번째 조건식이 참이면 실행하는 명령문 
}
else if ( 세번째 조건식 ) 
{ 
   세번째 조건식이 참이면 실행하는 명령문 
} 
... 
else 
{ 
   앞에 제시된 모든 조건식이 거짓이면 실행하는 명령문 
}


else 
  앞에 제시된 조건식이 거짓이면 해당 명령문을 실행하는 선택문 
if ( 조건식 ) 
{ 
   조건식이 참이면 실행하는 명령문 
} 
else 
{ 
   앞에 제시된 조건식이 거짓이면 실행하는 명령문 
}
