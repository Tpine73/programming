박코치의 C 프로그래밍


문제

점수에 따라 다른 메시지를 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 참고로, 90점이상이면 와! 끝내주게 잘 했다를 출력하고, 
80점이상이면 끝내주게 잘 했다를 출력하고, 
70점이상이면 잘 했다를 출력하고, 
나머지는 했다를 출력한다.

입출력 예제

0점 ~ 100점 사이의 점수를 입력하세요: 85
끝내주게 잘 했다


0점 ~ 100점 사이의 점수를 입력하세요: 93
와! 끝내주게 잘 했다


0점 ~ 100점 사이의 점수를 입력하세요: 72
잘 했다

알고리즘

프로그램 시작

   사용자에게 점수 입력받기

   90점이상이면 와! 끝내주게 잘 했다를 출력

   80점이상이면 끝내주게 잘 했다를 출력

   70점이상이면 잘 했다를 출력

   나머지는 했다를 출력

키워드 설명

		

int 
  -2, -1, 0, 1, 2 등의 정수 자료형


switch 
  정수형 변수가 특정 정수값이면 해당 명령문을 실행하는 선택문
switch( 정수형 변수 ) 
{ 
case 첫번째 정수값:
   변수가 첫번째 정수값이면 실행하는 명령문
case 두번째 정수값:
   변수가 두번째 정수값이면 실행하는 명령문
...
default:
   변수가 앞에서 제시된 정수값이 아니면 실행하는 명령문
}


/ 
  나눗셈 연산자


case 
  switch 문에서 정수형 변수를 특정 정수값과 비교하는 명령문


: 
  case문에서 정수형 변수와 특정 정수값을 비교할 때 사용되는 기호


default 
  switch 문에서 정수형 변수의 제시된 정수값이 없을 때를 표현
