박코치의 C 프로그래밍


문제

A양과 B군이 가위바위보를 선택하면 승패를 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요

입출력 예제

A양, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 2
바위를 냈습니다.
B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 1
가위를 냈습니다.
A양이 이겼습니다.


A양, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 3
보를 냈습니다.
B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 1
가위를 냈습니다.
B군이 이겼습니다.


A양, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 2
바위를 냈습니다.
B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 2
바위를 냈습니다.
A양과 B군이 비겼습니다.

알고리즘

프로그램시작

   A양이 가위바위보중 하나를 선택

   B군이 가위바위보중 하나를 선택

   A양과 B군의 가위바위보 승패 출력

      A양과 B군이 같으면 비김

      A양이 가위이고 B군이 보를 낸 경우 A양이 승리

      A양이 바위이고 B군이 가위를 낸 경우 A양이 승리

      A양이 보이고 B군이 바위를 낸 경우 A양이 승리

      나머지는 B군이 승리

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter03/RockPaperScissors.c 

	#include <stdio.h>  

 

	// 프로그램시작 

1	int main() 

	{ 

		// 변수 초기화 

		int playerA = 0; 

2		int playerB = 0; 

	 

		// A양이 가위바위보중 하나를 선택 

3		printf( "A양, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: " ); 

		// A양이 2를 선택한다고 가정 

4		scanf( "%d", &playerA ); 

	 

		// A양이 선택한 결과를 출력   

5		switch( playerA )								 

		{ 

6		case 1:										 

7			printf( "가위를 냈습니다.\n" );  

8			break;									 

9		case 2:										 

10			printf( "바위를 냈습니다.\n" );  

11			break;									 

12		case 3:										 

13			printf( "보를 냈습니다.\n" );  

14			break;									 

15		default:									 

16			printf( "가위바위보를 잘못 냈습니다.\n" );  

		} 

	 

		// B군이 가위바위보중 하나를 선택 

17		printf("B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: "); 

		// B군이 1을 선택한다고 가정 

18		scanf( "%d", &playerB ); 

	 

		// B군이 선택한 결과를 출력  

19		switch( playerB )								 

		{ 

20		case 1:										 

21			printf( "가위를 냈습니다.\n" ); 	 

22			break;									 

23		case 2:										 

24			printf( "바위를 냈습니다.\n" );  

25			break;									 

26		case 3:										 

27			printf( "보를 냈습니다.\n" );  

28			break;									 

29		default:									 

30			printf( "가위바위보를 잘못 냈습니다.\n" );  

		} 

	 

		// A양과 B군의 가위바위보 승패 출력 

		// A양과 B군이 같으면 비김  

31		if ( playerA == playerB )							 

		{ 

32			printf( "\nA양과 B군이 비겼습니다.\n" );  

		} 

		// A양이 가위이고 B군이 보를 낸 경우 A양이 승리 

33		else if ( ( playerA == 1 ) && ( playerB == 3 ) )				 

		{	 

34			printf( "\nA양이 이겼습니다.\n" );  

		} 

		// A양이 바위이고 B군이 가위를 낸 경우 A양이 승리 

35		else if ( ( playerA == 2 ) && ( playerB == 1 ) )				 

		{ 

36			printf( "\nA양이 이겼습니다.\n" );  

		} 

		// A양이 보이고 B군이 바위를 낸 경우 A양이 승리 

37		else if ( ( playerA == 3 ) && ( playerB == 2 ) )				 

		{ 

38			printf( "\nA양이 이겼습니다.\n" );  

		} 

		// 나머지는 B군이 승리  

39		else										 

		{ 

40			printf( "\nB군이 이겼습니다.\n" );  

		} 

	 

		// 프로그램 종료 

41		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

int 
  -2, -1, 0, 1, 2 등의 정수 자료형


switch 
  정수형 변수가 특정 정수값이면 해당 명령문을 실행하는 선택문
switch( 정수형 변수 ) 
{ 
case 첫번째 정수값:
   변수가 첫번째 정수값이면 실행하는 명령문
case 두번째 정수값:
   변수가 두번째 정수값이면 실행하는 명령문
...
default:
   변수가 앞에서 제시된 정수값이 아니면 실행하는 명령문
}


case 
  switch 문에서 정수형 변수를 특정 정수값과 비교하는 명령문


break 
  실행흐름을 중단하고 해당 영역을 빠져나오는 명령문으로, switch 문에서 break를 실행하면 이후 명령문은 건너띄고 switch 문을 빠져나옴 
switch( 정수형 변수 ) 
{ 
case 첫번째 정수값:
   변수가 첫번째 정수값이면 실행하는 명령문
   break;
case 두번째 정수값:
   변수가 두번째 정수값이면 실행하는 명령문
   break;
...
default:
   변수가 앞에서 제시된 정수값이 아니면 실행하는 명령문
}


default 
  switch 문에서 정수형 변수의 제시된 정수값이 없을 때를 표현


if 
  조건식이 참이면 해당 명령문을 실행하는 선택문
if ( 조건식 ) 
{ 
   조건식이 참이면 실행하는 명령문 
} 


== 
  등가연산자로 ( 왼쪽값 == 오른쪽값 )은 두 값이 같으면 참(1)을 반환하고 다르면 거짓(0)을 반환하며, 대입연산자 =와 비교하여 주의


else if 
  첫번째 조건식이 거짓이고 이후 조건식이 참이면 해당 명령문을 실행하는 선택문
if ( 첫번째 조건식 ) 
{ 
   첫번째 조건식이 참이면 실행하는 명령문 
} 
else if ( 두번째 조건식 ) 
{ 
   두번째 조건식이 참이면 실행하는 명령문 
}
else if ( 세번째 조건식 ) 
{ 
   세번째 조건식이 참이면 실행하는 명령문 
} 
... 
else 
{ 
   앞에 제시된 모든 조건식이 거짓이면 실행하는 명령문 
}


&& 
  논리연산자 AND로 ( 왼쪽조건식 && 오른쪽조건식 )은 두 조건식 모두 참이면 참(1)을 반환하고, 하나라도 거짓이면 거짓(0)을 반환


else 
  앞에 제시된 조건식이 거짓이면 해당 명령문을 실행하는 선택문 
if ( 조건식 ) 
{ 
   조건식이 참이면 실행하는 명령문 
} 
else 
{ 
   앞에 제시된 조건식이 거짓이면 실행하는 명령문 
}
