박코치의 C 프로그래밍


문제

입력받은 달의 영어이름을 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 참고로, 달의 영어이름은 
1월 - January, 
2월 - February, 
3월 - March, 
4월 - April, 
5월 - May, 
6월 - June, 
7월 - July, 
8월 - August, 
9월 - September, 
10월 - October, 
11월 - November, 
12월 - December와 같습니다.

입출력 예제

1월 ~ 12월 사이의 월을 입력하세요 : 4
4월의 영어이름은 April 입니다


1월 ~ 12월 사이의 월을 입력하세요 : 3
3월의 영어이름은 March 입니다


1월 ~ 12월 사이의 월을 입력하세요 : 5
5월의 영어이름은 May 입니다

키워드 설명

		

int 
  -2, -1, 0, 1, 2 등의 정수 자료형


switch 
  정수형 변수가 특정 정수값이면 해당 명령문을 실행하는 선택문
switch( 정수형 변수 ) 
{ 
case 첫번째 정수값:
   변수가 첫번째 정수값이면 실행하는 명령문
case 두번째 정수값:
   변수가 두번째 정수값이면 실행하는 명령문
...
default:
   변수가 앞에서 제시된 정수값이 아니면 실행하는 명령문
}


case 
  switch 문에서 정수형 변수를 특정 정수값과 비교하는 명령문


: 
  case문에서 정수형 변수와 특정 정수값을 비교할 때 사용되는 기호


break 
  실행흐름을 중단하고 해당 영역을 빠져나오는 명령문으로, switch 문에서 break를 실행하면 이후 명령문은 건너띄고 switch 문을 빠져나옴 
switch( 정수형 변수 ) 
{ 
case 첫번째 정수값:
   변수가 첫번째 정수값이면 실행하는 명령문
   break;
case 두번째 정수값:
   변수가 두번째 정수값이면 실행하는 명령문
   break;
...
default:
   변수가 앞에서 제시된 정수값이 아니면 실행하는 명령문
}


default 
  switch 문에서 정수형 변수의 제시된 정수값이 없을 때를 표현
