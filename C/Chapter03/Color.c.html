문제

사용자가 좋아하는 색상을 선택하면 색깔 심리테스트 결과를 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 참고로, 색상별 심리테스트 결과는 아래와 같습니다. 

* 빨강색을 선택한 당신은 진취적이고 열정적이며 개방적이고 자신감이 넘치는군요

* 노랑색을 선택한 당신은 밝고 솔직하고 명랑하며 호기심도 강하고 주의력과 집중력이 높네요

* 초록색을 선택한 당신은 성실하고 참을성이 많으며 충성심이 강하고 의리가 있군요

* 파랑색을 선택한 당신은 차분하고 침착하고 이성적이며 어떤 일이든 진지하게 열심히 하는군요

* 보라색을 선택한 당신은 섬세하고 감수성이 풍부하며 관찰력이 있고 예술적 재능이 뛰어나군요

입출력 예제

빨강(1), 노랑(2), 초록(3), 파랑(4), 보라(5) 중 하나를 숫자로 입력하세요: 4
파랑색을 선택한 당신은 차분하고 침착하고 이성적이며 어떤 일이든 진지하게 열심히 하는군요


빨강(1), 노랑(2), 초록(3), 파랑(4), 보라(5) 중 하나를 숫자로 입력하세요: 2
노랑색을 선택한 당신은 밝고 솔직하고 명랑하며 호기심도 강하고 주의력과 집중력이 높네요


빨강(1), 노랑(2), 초록(3), 파랑(4), 보라(5) 중 하나를 숫자로 입력하세요: 3
초록색을 선택한 당신은 성실하고 참을성이 많으며 충성심이 강하고 의리가 있군요

프로그램 시작

   사용자에게 색깔 입력받기

   재미로 보는 색깔 심리테스트 결과 출력

      빨강을 선택하면 '진취적이고 열정적이며 개방적이고 자신감이 넘치는군요' 출력

      노랑을 선택하면 '밝고 솔직하고 명랑하며 호기심도 강하고 주의력과 집중력이 높네요' 출력

      초록을 선택하면 '성실하고 참을성이 많으며 충성심이 강하고 의리가 있군요' 출력

      파랑을 선택하면 '차분하고 침착하고 이성적이며 어떤 일이든 진지하게 열심히 하는군요' 출력

      보라를 선택하면 '섬세하고 감수성이 풍부하며 관찰력이 있고 예술적 재능이 뛰어나군요' 출력

키워드 설명

		

int 
  -2, -1, 0, 1, 2 등의 정수 자료형


switch 
  정수형 변수가 특정 정수값이면 해당 명령문을 실행하는 선택문
switch( 정수형 변수 ) 
{ 
case 첫번째 정수값:
   변수가 첫번째 정수값이면 실행하는 명령문
case 두번째 정수값:
   변수가 두번째 정수값이면 실행하는 명령문
...
default:
   변수가 앞에서 제시된 정수값이 아니면 실행하는 명령문
}


case 
  switch 문에서 정수형 변수를 특정 정수값과 비교하는 명령문


: 
  case문에서 정수형 변수와 특정 정수값을 비교할 때 사용되는 기호


\033 
  출력 설정 이스케이프 문자 
printf( "\033[31m 글자색 빨강 \033[0m\n" ); // 출력결과: 글자색 빨강 
printf( "\033[33m 글자색 노랑 \033[0m\n" ); // 출력결과:  글자색 노랑 
printf( "\033[32m 글자색 초록 \033[0m\n" ); // 출력결과:  글자색 초록 
printf( "\033[34m 글자색 파랑 \033[0m\n" ); // 출력결과:  글자색 파랑 
printf( "\033[41m 바탕색 빨강 \033[0m\n" ); // 출력결과:  바탕색 빨강  
printf( "\033[43m 바탕색 노랑 \033[0m\n" ); // 출력결과:  바탕색 노랑 
printf( "\033[42m 바탕색 초록 \033[0m\n" ); // 출력결과:  바탕색 초록 
printf( "\033[44m 바탕색 파랑 \033[0m\n" ); // 출력결과:  바탕색 파랑 
printf( "\033[01m 글자 굵게    \033[0m\n" );   // 출력결과:  글자 굵게 
 ※ \033[0m는 글자색이나 배경색 초기화
 ※ \033[H\033[J는 모니터 화면을 깨끗이 지우고 초기화


break 
  실행흐름을 중단하고 해당 영역을 빠져나오는 명령문으로, switch 문에서 break를 실행하면 이후 명령문은 건너띄고 switch 문을 빠져나옴 
switch( 정수형 변수 ) 
{ 
case 첫번째 정수값:
   변수가 첫번째 정수값이면 실행하는 명령문
   break;
case 두번째 정수값:
   변수가 두번째 정수값이면 실행하는 명령문
   break;
...
default:
   변수가 앞에서 제시된 정수값이 아니면 실행하는 명령문
}


default 
  switch 문에서 정수형 변수의 제시된 정수값이 없을 때를 표현
