박코치의 C 프로그래밍


문제

현재 기온을 입력하면 그 결과를 바탕으로 한파 경보, 한파 주의보, 폭염 주의보, 폭염 경보를 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 참고로, 현재 기온이 
-15도 이하이면 한파 경보를 발효하고,
-13도 이하이면 한파 주의보를 발효하고,
35도 이상이면 폭염 경보를 발효하고,
33도 이상이면 폭염 주의보를 발효한다고 가정합니다.

입출력 예제

섭씨 -100도부터 100도 사이에서 현재 기온은 몇 도인가요 : 18
오늘 날씨는 그나마 살만하네요


섭씨 -100도부터 100도 사이에서 현재 기온은 몇 도인가요 : -14
일부 지방에 한파 주의보가 발효중입니다.


섭씨 -100도부터 100도 사이에서 현재 기온은 몇 도인가요 : 35
일부 지방에 폭염 경보가 발효중입니다.

알고리즘

프로그램 시작

   현재 기온을 입력받기

   현재 기온이 한파 경보, 한파 주의보, 폭염 경보, 폭염 주의보 중 무엇인지를 출력하기

      현재 기온이 -15도 이하이면 한파 경보 발효

      현재 기온이 -13도 이하이면 한파 주의보 발효

      현재 기온이 35도 이상이면 폭염 경보 발효

      현재 기온이 33도 이상이면 폭염 주의보 발효

      나머지의 경우 그나마 살만하다고 출력하기

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter03/Weather.c 

	#include <stdio.h>  

 

	// 프로그램 시작 

1	int main() 

	{ 

2		int temperature = 0; 

 

		// 현재 기온을 입력받기  

3		printf("섭씨 -100도부터 100도 사이에서 현재 기온은 몇 도인가요 : " ); 

		// 18도를 입력한다고 가정 

4		scanf( "%d", &temperature ); 

	 

		// 현재 기온이 한파 경보, 한파 주의보, 폭염 경보, 폭염 주의보 중 무엇인지를 출력하기 

		// 현재 기온이 -15도 이하이면 한파 경보 발효 

5		if (  temperature <= -15 )								 

		{ 

6			printf( "일부 지방에 한파 경보가 발효중입니다. " ); 

		} 

		// 현재 기온이 -13도 이하이면 한파 주의보 발효  

7		else if (  ( -15 < temperature ) && ( temperature <= -13 ) )				 

		{ 

8			printf( "일부 지방에 한파 주의보가 발효중입니다. " ); 

		} 

		// 현재 기온이 35도 이상이면 폭염 경보 발효  

9		else if ( 35 <= temperature )							 

		{ 

10			printf( "일부 지방에 폭염 경보가 발효중입니다. " ); 

		} 

		// 현재 기온이 33도 이상이면 폭염 주의보 발효  

11		else if (  ( 33 <= temperature ) && ( temperature < 35 ) )				 

		{ 

12			printf( "일부 지방에 폭염 주의보가 발효중입니다. " ); 

		} 

		// 나머지의 경우 그나마 살만하다고 출력하기 

13		else 											 

		{ 

14			printf( "오늘 날씨는 그나마 살만하네요 " ); 

		} 

		 

		// 프로그램 종료 

15		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

if 
  조건식이 참이면 해당 명령문을 실행하는 선택문
if ( 조건식 ) 
{ 
   조건식이 참이면 실행하는 명령문 
} 


<= 
  비교연산자로 ( 왼쪽값 <= 오른쪽값 )은 왼쪽값이 오른쪽값보다 작거나 같으면 참(1)을 반환하고 아니면 거짓(0)을 반환


else if 
  첫번째 조건식이 거짓이고 이후 조건식이 참이면 해당 명령문을 실행하는 선택문
if ( 첫번째 조건식 ) 
{ 
   첫번째 조건식이 참이면 실행하는 명령문 
} 
else if ( 두번째 조건식 ) 
{ 
   두번째 조건식이 참이면 실행하는 명령문 
}
else if ( 세번째 조건식 ) 
{ 
   세번째 조건식이 참이면 실행하는 명령문 
} 
... 
else 
{ 
   앞에 제시된 모든 조건식이 거짓이면 실행하는 명령문 
}


&& 
  논리연산자 AND로 ( 왼쪽조건식 && 오른쪽조건식 )은 두 조건식 모두 참이면 참(1)을 반환하고, 하나라도 거짓이면 거짓(0)을 반환


else 
  앞에 제시된 조건식이 거짓이면 해당 명령문을 실행하는 선택문 
if ( 조건식 ) 
{ 
   조건식이 참이면 실행하는 명령문 
} 
else 
{ 
   앞에 제시된 조건식이 거짓이면 실행하는 명령문 
}
