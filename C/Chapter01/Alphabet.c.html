박코치의 C 프로그래밍


문제

알파벳을 입력받아 그 값을 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요

입출력 예제

알파벳 한 글자를 입력하세요: k
k를 입력했군요^^


알파벳 한 글자를 입력하세요: j
j를 입력했군요^^


알파벳 한 글자를 입력하세요: A
A를 입력했군요^^

알고리즘

프로그램 시작

   알파벳을 입력받기

   입력값 출력하기

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter01/Alphabet.c 

	#include <stdio.h> 					 

	 

	// 프로그램 시작 

1	int main() 

	{ 

		// 알파벳 변수 초기화 

2		char letter = '\0'; 

	 

		// 알파벳을 입력받기  

3		printf("알파벳 한 글자를 입력하세요: " );	 

		// k를 입력한다고 가정 

4		scanf( "%c", &letter );				 

	 

		// 입력값 출력하기 

5		printf( "%c를 입력했군요^^", letter );		 

	 

		// 프로그램 종료 

6		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

include 
  헤더파일(.h)을 소스코드(.c)에 포함하는 명령문


stdio.h 
  standard input output의 줄임말로 printf나 scanf 등 표준입출력 관련 함수 정보를 가지고 있는 헤더파일


printf 
  print formatted의 줄임말로 전달받은 내용을 형식에 맞게 모니터에 출력하는 함수


scanf 
  scan formatted의 줄임말로 키보드에서 형식에 맞게 입력받은 값을 가져오는 함수


%c 
  1글자의 문자 형식으로 입출력
