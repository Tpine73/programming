키워드 설명

		

include 
  헤더파일(.h)을 소스코드(.c)에 포함하는 명령문


stdio.h 
  standard input output의 줄임말로 printf나 scanf 등 표준입출력 관련 함수 정보를 가지고 있는 헤더파일


main 
  프로그램을 시작할 때 운영체제가 호출하는 함수


printf 
  print formatted의 줄임말로 전달받은 내용을 형식에 맞게 모니터에 출력하는 함수


\n 
  줄바꿈(new line) 문자


return 
  함수 종료시 실행흐름이 함수를 호출했던 곳으로 되돌아감 
예를 들어, main 함수가 끝나면 main 함수를 호출한 운영체제로 제어흐름이 돌아가고 프로그램이 종료
