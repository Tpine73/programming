A양, 가위, 바위, 보 중 하나를 선택하세요: 바위
바위를 냈습니다.
B군, 가위, 바위, 보 중 하나를 선택하세요: 가위
가위를 냈습니다.
A양이 이겼습니다.


A양, 가위, 바위, 보 중 하나를 선택하세요: 보
보를 냈습니다.
B군, 가위, 바위, 보 중 하나를 선택하세요: 가위
가위를 냈습니다.
B군이 이겼습니다.

바위를 냈습니다.
B군, 가위, 바위, 보 중 하나를 선택하세요: 바위

A양과 B군이 비겼습니다.

알고리즘

프로그램시작

   A양이 가위바위보중 하나를 선택

   B군이 가위바위보중 하나를 선택

   A양과 B군의 가위바위보 승패 출력

      A양과 B군이 같으면 비김

      A양이 가위이고 B군이 보를 낸 경우 A양 승리

      A양이 바위이고 B군이 가위를 낸 경우 A양 승리

      A양이 보이고 B군이 바위를 낸 경우 A양 승리

      나머지 경우는 B군이 승리

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter05/RockPaperScissors.c 

	#include <stdio.h> 		       

	#include <string.h> 								 

	 

	// 프로그램시작 

1	int main() 

	{ 

		// 변수 초기화 

		char playerA[10] = "";  

2		char playerB[10] = "";  

	 

		// A양이 가위바위보중 하나를 선택 

		printf("A양, 가위, 바위, 보 중 하나를 선택하세요: "); 

		// A양이 바위를 선택한다고 가정 

		scanf( "%s", playerA );							 

3		printf( "%s를 냈습니다.\n", playerA );  

	 

		// B군이 가위바위보중 하나를 선택 

		printf("B군, 가위, 바위, 보 중 하나를 선택하세요: "); 

		// B군이 가위를 선택한다고 가정 

		scanf( "%s", playerB );							 

4		printf( "%s를 냈습니다.\n", playerB );  

	 

		// A양과 B군의 가위바위보 승패 출력 

		// A양과 B군이 같으면 비김  

5		if ( !strcmp( playerA, playerB ) )					 

		{ 

6			printf( "\nA양과 B군이 비겼습니다.\n" );  

		} 

		// A양이 가위이고 B군이 보를 낸 경우 A양 승리 

7		else if ( !strcmp( playerA, "가위" ) && !strcmp( playerB, "보" ) )	 

		{	 

8			printf( "\nA양이 이겼습니다.\n" );  

		} 

		// A양이 바위이고 B군이 가위를 낸 경우 A양 승리 

9		else if ( !strcmp( playerA, "바위" ) && !strcmp( playerB, "가위" ) )	 

		{ 

10			printf( "\nA양이 이겼습니다.\n" );  

		} 

		// A양이 보이고 B군이 바위를 낸 경우 A양 승리 

11		else if ( !strcmp( playerA, "보" ) && !strcmp( playerB, "바위" ) )	 

		{ 

12			printf( "\nA양이 이겼습니다.\n" );  

		} 

		// 나머지 경우는 B군이 승리  

13		else 

		{ 

14			printf( "\nB군이 이겼습니다.\n" );  

		} 

	 

		// 프로그램 종료 

15		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

string.h 
  문자열 관련 함수 정보를 가지고 있는 헤더파일


%s 
  문자열  형식으로 입출력


! 
  NOT연산자. 문자열 비교 결과는 문자열이 같으면 0(거짓)을 반환하므로, 이를 1(참)으로 반전시키려고 NOT연산자를 사용


strcmp 
  string compare의 줄임말로, strcmp( 왼쪽문자열, 오른쪽문자열 )은 왼쪽문자열이 오른쪽문자열보다 알파벳순으로 앞이면 음수, 같으면 0, 뒤면 양수를 반환

 예를 들어, strcmp( "cat", "cat" )을 실행하면 두 문자열이 동일하므로 0을 반환하지만, strcmp( "cat", "dog" )를 실행하면 "cat"이 "dog" 보다 앞에 있으므로 음수를 반환
