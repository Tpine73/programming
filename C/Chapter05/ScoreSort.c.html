알고리즘

프로그램시작

   점수들 입력

   점수 정렬

      영역의 마지막 칸에 가장 큰 값을 저장하면서, 영역을 한 칸씩 앞으로 좁혀가기

         영역의 첫번째 칸부터 인접한 두 값을 비교하면서

         앞이 뒤보다 크면 두 값을 맞바꾸기하면서 뒤쪽에 큰 값을 저장

   정렬된 점수 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter05/ScoreSort.c 

	#include <stdio.h> 

	 

	// 프로그램시작 

1	int main() 

	{ 

		// 3칸짜리 배열을 선언  // [주석포함] 

		int score[3] = { 0 }; 

		int index = 0; 

		int max = 0; 

2		int temp = 0; 

		 

		// 점수들 입력  

3		for( index = 0; index < 3; index ++ ) 

		{ 

			printf( "0점 ~ 100점 사이의 점수를 입력하세요: " ); 

			// 90, 89, 86을 순서대로 입력한다고 가정 

4			scanf( "%d", &score[ index ] ); 

		} 

 

		// 점수 정렬 

		// 영역의 마지막 칸에 가장 큰 값을 저장하면서, 영역을 한 칸씩 앞으로 좁혀가기 

5		for ( max = 2; max >= 0; max -- ) 

		{ 

			// 영역의 첫번째 칸부터 인접한 두 값을 비교하면서  

6			for ( index = 0; index < max; index ++ )			 

			{ 

				// 앞이 뒤보다 크면 두 값을 맞바꾸기하면서 뒤쪽에 큰 값을 저장 

7				if ( score[ index ] > score[ index + 1 ] ) 

				{ 

8					temp = score[ index ]; 

9					score[ index ] = score[ index + 1 ];						 

10					score[ index + 1 ] = temp;							 

				} 

			} 

		} 

	 

		// 정렬된 점수 출력 

11		for ( index = 0; index < 3; index ++ ) 

		{ 

12			printf( "%d) %d ", index + 1, score[ index ] ); 

		} 

 

		// 프로그램 종료 

13		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

[3] 
  3칸짜리 배열을 선언


{ 
  여는 중괄호로 배열을 선언하면서 초기화할 때 사용


} 
  닫는 중괄호로 배열을 선언하면서 초기화할 때 사용


[ index ] 
  배열의 여러 칸 중에서 index번째 칸을 의미하며, 참고로 배열은 0번째 칸부터 시작


< 
  비교연산자로 ( 왼쪽값 < 오른쪽값 )은 왼쪽값이 오른쪽값보다 작으면 참(1)을 반환하고 아니면 거짓(0)을 반환


max 
  영역의 마지막 칸을 의미하는 변수


> 
  비교연산자로 ( 왼쪽값 > 오른쪽값 )는 왼쪽값이 오른쪽값보다 크면 참(1)을 반환하고 아니면 거짓(0)을 반환
