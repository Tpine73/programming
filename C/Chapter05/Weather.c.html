박코치의 C 프로그래밍


문제

기상 특보를 입력하면, 이를 바탕으로 주의사항을 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 참고로, 대설특보는 눈길 운전 조심을 출력하고, 호우특보는 빗길 운전 조심을 출력하고, 강풍주의보는 안전 운전 조심을 출력하세요.

입출력 예제

오늘의 기상 특보를 입력하세요 : 대설특보
일부 지방에 대설특보가 발효중입니다. 출근길 눈길 운전에 유의해야 합니다.


오늘의 기상 특보를 입력하세요 : 호우특보
일부 지방에 호우특보가 발효중입니다. 출근길 빗길 운전에 유의해야 합니다.


오늘의 기상 특보를 입력하세요 : 강풍주의보
일부 지방에 강풍주의보가 발효중입니다. 출근길 안전 운전에 유의해야 합니다.

알고리즘

프로그램 시작

   기상 특보를 입력받기

   날씨 뉴스 기사 생성

      대설특보는 눈길 운전 조심을 출력

      호우특보는 빗길 운전 조심을 출력

      강풍주의보는 안전 운전 조심을 출력

프로그램 코드 

	// 파일명 : ./Chapter05/Weather.c 

	#include <stdio.h>  

	#include <string.h> 								 

	 

	// 프로그램 시작 

1	int main() 

	{ 

		char weatherReport[32] = "";					 

2		char newsMessage[1024] = "";

3		scanf( "%s", weatherReport );

10			strcat( newsMessage, "안전 " );

키워드 설명

		

string.h 
  문자열 관련 함수 정보를 가지고 있는 헤더파일


"" 
  문자배열에서 빈문자열을 표현


strcpy 
  string copy의 줄임말로 strcpy( 문자열변수, 복사할 문자열 )은 문자열을 복사하는 함수

변수공간a원본 문자열와 변수공간b복사본 문자열에 대해서 strcpy( a, b )을 실행하면 변수공간a복사본 문자열로 변경



! 
  NOT연산자. 문자열 비교 결과는 문자열이 같으면 0(거짓)을 반환하므로, 이를 1(참)으로 반전시키려고 NOT연산자를 사용


strcmp 
  string compare의 줄임말로, strcmp( 왼쪽문자열, 오른쪽문자열 )은 왼쪽문자열이 오른쪽문자열보다 알파벳순으로 앞이면 음수, 같으면 0, 뒤면 양수를 반환

 예를 들어, strcmp( "cat", "cat" )을 실행하면 두 문자열이 동일하므로 0을 반환하지만, strcmp( "cat", "dog" )를 실행하면 "cat"이 "dog" 보다 앞에 있으므로 음수를 반환



strcat 
  string concatenate의 줄임말로 strcat( 대상문자열, 덧붙일 문자열 )은 문자열을 덧붙이는 함수

예를 들어 변수공간 a 원본 문자열와 변수공간b 덧붙일 문자열에 대해서 strcpy( a, b )을 실행하면 변수공간 a 원본 문자열 덧붙일 문자열로 변경
