프로그램시작

   단어들 입력

   단어 정렬

      영역의 마지막 칸에 가장 큰 값을 저장하면서, 영역을 한 칸씩 앞으로 좁혀가기

         첫 번째 칸 단어보다 알파벳순으로 더 앞에 있는 단어를 만나면 두 단어를 맞바꾸기

   정렬된 단어 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter05/WordSort.c 

 	#include <stdio.h> 

 	#include <string.h> 

	 

	// 프로그램시작 

1	int main() 

	{ 

		// 32글자 이내로 구성된 단어 3개를 저장할 수 있도록 32칸씩 3줄로 구성된 2차원 배열을 선언 // [주석포함]  

		char word[3][32] = { { '\0' }, { '\0' }, { '\0' } }; 

		// 한 칸당 최대 32byte 문자열을 저장할 수 있는 1차원 배열을 선언 // [주석포함] 

		char temp[32] = { '\0' }; 

		int index = 0; 

2		int max = 0; 

 

		 

		// 단어들 입력  

3		for( index = 0; index < 3; index ++ ) 								 

		{ 

			printf( "단어를 입력하세요: " ); 

			// 첫번째는 carrot, 두번째는 banana, 세번째는 apple을 입력한다고 가정 

4			scanf( "%s", word[ index ] ); 

		} 

 

		// 단어 정렬 

		// 영역의 마지막 칸에 가장 큰 값을 저장하면서, 영역을 한 칸씩 앞으로 좁혀가기 

5		for ( max = 2; max >= 0; max -- )			 

		{ 

			// 영역의 첫번째 칸부터 인접한 두 값을 비교하면서  

6			for ( index = 0; index < max; index ++ )			 

			{ 

				// 첫 번째 칸 단어보다 알파벳순으로 더 앞에 있는 단어를 만나면 두 단어를 맞바꾸기  

7				if ( strcmp( word[ index ], word[ index + 1 ] ) > 0 )			 

				{ 

8					strcpy( temp, word[ index ] );					 

9					strcpy( word[ index ], word[ index + 1 ] );				 

10					strcpy( word[ index + 1 ], temp );						 

				} 

			} 

		} 

	 

		// 정렬된 단어 출력 

11		for ( index = 0; index < 3; index ++ )  							 

		{ 

12			printf( "%d) %s ", index + 1, word[ index ] ); 

		} 

	 

		// 프로그램 종료 

13		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

[3][32] 
  32글자 이내로 구성된 단어 3개를 저장할 수 있도록 32칸씩 3줄로 구성된 2차원 배열을 선언


[32] 
  한 칸당 최대 32byte 문자열을 저장할 수 있는 1차원 배열을 선언


{ 
  여는 중괄호로 배열을 선언하면서 초기화할 때 사용


'\0' 
  빈문자(NULL문자)


} 
  닫는 중괄호로 배열을 선언하면서 초기화할 때 사용


[ index ] 
  배열의 여러 칸 중에서 index번째 칸을 의미하며, 참고로 배열은 0번째 칸부터 시작


strcmp 
  string compare의 줄임말로, strcmp( 왼쪽문자열, 오른쪽문자열 )은 왼쪽문자열이 오른쪽문자열보다 알파벳순으로 앞이면 음수, 같으면 0, 뒤면 양수를 반환

 예를 들어, strcmp( "cat", "cat" )을 실행하면 두 문자열이 동일하므로 0을 반환하지만, strcmp( "cat", "dog" )를 실행하면 "cat"이 "dog" 보다 앞에 있으므로 음수를 반환



strcpy 
  string copy의 줄임말로 strcpy( 문자열변수, 복사할 문자열 )은 문자열을 복사하는 함수

변수공간a원본 문자열와 변수공간b복사본 문자열에 대해서 strcpy( a, b )을 실행하면 변수공간a복사본 문자열로 변경
