박코치의 C 프로그래밍


문제

사용자가 입력한 두 이름이 같은 성인지 다른 성인지 판단하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요

입출력 예제

A양, 이름을 입력하세요: 허난설헌
B군, 이름을 입력하세요: 허균
허난설헌과 허균은 성이 같습니다.


A양, 이름을 입력하세요: 신사임당
B군, 이름을 입력하세요: 왕건
신사임당과 왕건은 성이 다릅니다.


A양, 이름을 입력하세요: 김만덕
B군, 이름을 입력하세요: 김유신
김만덕과 김유신은 성이 같습니다.

알고리즘

프로그램시작

   이름 2개를 입력

   이름에서 첫 음절(2byte)을 비교하기

      첫 음절이 같으면 성이 같습니다 출력

      첫 음절이 다르면 성이 다릅니다 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter05/FamilyName.c 

 	#include <stdio.h> 

 	#include <string.h> 				 

	 

	// 프로그램시작 

1	int main() 

	{ 

		// 한 칸당 최대 10byte 문자열을 저장할 수 있는 배열을 선언  // [주석포함]  

		char nameA[10] = { '\0' };					 

2		char nameB[10] = { '\0' };					 

		 

		// 이름 2개를 입력  

		printf( "A양, 이름을 입력하세요: " ); 

		// 허    난    설    헌   \0을 입력한다고 가정 

3		scanf( "%s", nameA ); 

		printf( "B군, 이름을 입력하세요: " ); 

		// 허    균   \0을 입력한다고 가정 

4		scanf( "%s", nameB ); 

	 

		// 이름에서 첫 음절(2byte)을 비교하기 

		// 첫 음절이 같으면 성이 같습니다 출력 

5		if ( !strncmp( nameA, nameB, 2 ) )				 

		{ 

6			printf( "%s과 %s은 성이 같습니다. ", nameA, nameB ); 

		} 

		// 첫 음절이 다르면 성이 다릅니다 출력 

7		else 

		{ 

8			printf( "%s과 %s은 성이 다릅니다. ", nameA, nameB ); 

		} 

		 

		// 프로그램 종료 

9		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

string.h 
  문자열 관련 함수 정보를 가지고 있는 헤더파일


[10] 
  한 칸당 최대 10byte 문자열을 저장할 수 있는 1차원 배열을 선언


{ 
  여는 중괄호로 배열을 선언하면서 초기화할 때 사용


'\0' 
  빈문자(NULL문자)


} 
  닫는 중괄호로 배열을 선언하면서 초기화할 때 사용


! 
  NOT연산자. 문자열 비교 결과는 문자열이 같으면 0(거짓)을 반환하므로, 이를 1(참)으로 반전시키려고 NOT연산자를 사용


strncmp 
  string number compare의 줄임말로, strcnmp( 왼쪽문자열, 오른쪽문자열, 비교길이 )는 왼쪽문자열과 오른쪽문자열을 주어진 길이만큼만 비교하며 왼쪽문자열이 오른쪽문자열보다 알파벳순으로 앞이면 음수, 같으면 0, 뒤면 양수를 반환

 예를 들어, strncmp( "decimal", "decision", 4 )을 실행하면 4byte "deci"가 동일하므로 0을 반환하지만, strncmp( "decimal", "decision", 7 )를 실행하면 7byte "decimal"가 "decisio"보다 앞에 있으므로 음수를 반환



2 
  왼쪽문자열과 오른쪽문자열의 한글 1글자(2byte)만큼만 비교한다는 의미
