알고리즘

프로그램시작

   예약 좌석은 (예약)으로 출력하면서

      전체 좌석에 대한 현재 예약 상태 출력

   사용자에게 예약 좌석 입력 받기

   입력 좌석이 유효하면 좌석 예약 및 출력을 반복

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter05/TrainSeat.c 

	#include <stdio.h> 

	#include <string.h> 				 

	 

	// 프로그램시작 

1	int main() 

	{ 

		// 한 칸당 최대 32byte 문자열을 저장할 수 있는 5칸씩 4줄로 구성된 3차원 배열을 선언 및 초기화 // [주석포함]  

		char seat[4][5][32] = { '\0' }; 

		// 한 칸당 최대 32byte 문자열을 저장할 수 있는 1차원 배열을 선언 // [주석포함] 

2		char reservedSeat[32] = "A1"; 

	 

3		do 

		{ 

		  // 예약 좌석은 (예약)으로 출력하면서 

4		  strcpy( seat[ (int)( reservedSeat[0] - 'A' ) ][ (int)( reservedSeat[1] - '1' ) ], "(예약)" );	 

	 

		  // 전체 좌석에 대한 현재 예약 상태 출력 

		  printf( "#   %d %6d %6d %6d %6d\n", 1, 2, 3, 4, 5 );  

		  printf( "A %6s %6s %6s %6s %6s\n", seat[0][0], seat[0][1], seat[0][2], seat[0][3], seat[0][4] ); 

		  printf( "B %6s %6s %6s %6s %6s\n", seat[1][0], seat[1][1], seat[1][2], seat[1][3], seat[1][4] ); 

		  printf( "C %6s %6s %6s %6s %6s\n", seat[2][0], seat[2][1], seat[2][2], seat[2][3], seat[2][4] ); 

5		  printf( "D %6s %6s %6s %6s %6s\n", seat[3][0], seat[3][1], seat[3][2], seat[3][3], seat[3][4] ); 

			 

		  // 사용자에게 예약 좌석 입력 받기 

		  printf( "A2, B4, 0(종료)와 같이, 예약 희망 좌석을 입력하세요 : " ); 

		  // 첫번째는 A2, 두번째는 0을 입력한다고 가정 

6		  scanf( "%s", reservedSeat );  

	 

		// 입력 좌석이 유효하면 좌석 예약 및 출력을 반복 

		} while( ( 'A' <= reservedSeat[0] ) && ( reservedSeat[0] <= 'D' ) 

7		    && ( '1' <= reservedSeat[1] ) && ( reservedSeat[1] <= '5' ) ); 

		 

		// 프로그램 종료 

8		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

string.h 
  문자열 관련 함수 정보를 가지고 있는 헤더파일


[4][5][32] 
  한 칸당 최대 32byte 문자열을 저장할 수 있는 5칸씩 4줄로 구성된 3차원 배열을 선언


[32] 
  한 칸당 최대 32byte 문자열을 저장할 수 있는 1차원 배열을 선언


{ 
  여는 중괄호로 배열을 선언하면서 초기화할 때 사용


'\0' 
  빈문자(NULL문자)


} 
  닫는 중괄호로 배열을 선언하면서 초기화할 때 사용


strcpy 
  string copy의 줄임말로 strcpy( 문자열변수, 복사할 문자열 )은 문자열을 복사하는 함수

변수공간a원본 문자열와 변수공간b복사본 문자열에 대해서 strcpy( a, b )을 실행하면 변수공간a복사본 문자열로 변경



(int) 
  값의 자료형을 강제로 정수 자료형으로 형변환

예를 들어 a = (int)3.14 = 3 또는 b = (int)'A' = 65



'A' 
  seat 열 인덱스가 A열=0(='A'-'A'), B열=1(='B'-'A'), C열=2(='C'-'A'), D열=3(='D'-'A')에 매핑되도록 보정하는 값


'1' 
  seat 행 인덱스가 1행=0(='1'-'1'), 2행=1(='2'-'1'), 3행=2(='3'-'1'), 4행=3(='4'-'1')에 매핑되도록 보정하는 값
