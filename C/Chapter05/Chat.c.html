프로그램 시작

   사용자와 시스템의 대화 반복

      사용자 입력

      시스템 답변 출력

         준비한 대화쌍에서

            적절한 답변을 찾으면 시스템 답변을 출력

            적절한 답변을 못 찾으면 '뭐 딴거 재밌는거 없을까?' 출력

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter05/Chat.c 

	#include <stdio.h> 

	#include <string.h>					 

 

	// 프로그램 시작 	

1	int main( void ) 

	{ 

		int	index = 0; 

		// 한 칸당 최대 128byte 문자열을 저장할 수 있는 1차원 배열을 선언 // [주석포함] 

		char	input[128] = "\0"; 

		// 한 칸당 최대 128byte 문자열을 저장할 수 있는 2칸씩 4줄로 구성된 3차원 배열을 선언 // [주석포함] 

		char	dialogue[4][2][128] =   

			{ { "이름이 뭐", "난 재석이야. 국민MC 유재석과 이름이 같아. ㅎㅎㅎ 넌 몇 살이니?" }, 

			  { "몇 살", "난 20살이야. 남들이 다 부러워하는 나이지. 넌 어디 살아?" }, 

			  { "어디 살", "난 홍지동 살아. 북한산 아래 있어서 공기가 무지 맑아.ㅎㅎㅎ 취미가 뭐니?" }, 

			  { "만나서 반가", "나도 반가워 친하게 지내자" } }; 

	 

		// 사용자와 시스템의 대화 반복 

2		printf( "[시스템] 안녕? 넌 이름이 모야?\n 사용자: " ); 

3		while( 1 ) 

		{ 

			// 사용자 입력 	

			// 첫번째는 '난 희철 만나서 반가워', 두번째는 '앗! 미안 지금 갑자기 나가봐야 해~ 다음에 또 봐 바이'라고 입력한다고 가정  

4			fgets( input, 128, stdin ); 			 

	 

			// 시스템 답변 출력 			

			// 사용자가 '바이'를 입력하면 '그래 안녕! 담에 또 봐~'를 출력 후 대화를 종료  

5			if ( strstr( input, "바이" ) != NULL ) 		 

			{ 

				printf( "[시스템] 그래 안녕! 담에 또 봐~\n" ); 

6				break; 

			} 

 

			// 준비한 대화쌍에서 

7			for ( index = 0; index < 4; index++ ) 

			{ 

				// 적절한 답변을 찾으면 시스템 답변을 출력 

8				if ( strstr( input, dialogue[index][0] ) != NULL ) 		 

				{ 

					printf( "[시스템] %s\n 사용자: ", dialogue[index][1] ); 	 

9					break; 

				} 

			} 

			 

			// 적절한 답변을 못 찾으면 '뭐 딴거 재밌는거 없을까?' 출력 

10			if ( index == 4 ) 

			{ 

11				printf( "[시스템] 뭐 딴거 재밌는거 없을까?\n 사용자: " ); 

			} 

		} 

 

		// 프로그램 종료 	

12		return 0; 

	} 

		
		

	
	

	

		
키워드 설명

		

string.h 
  문자열 관련 함수 정보를 가지고 있는 헤더파일


[128] 
  한 칸당 최대 128byte 문자열을 저장할 수 있는 1차원 배열을 선언


" 
  문자열의 시작과 끝을 표현하는 기호


[4][2][128] 
  한 칸당 최대 128byte 문자열을 저장할 수 있는 2칸씩 4줄로 구성된 3차원 배열을 선언


{ 
  여는 중괄호로 배열을 선언하면서 초기화할 때 사용


} 
  닫는 중괄호로 배열을 선언하면서 초기화할 때 사용


fgets 
  file get string의 줄임말로 fgets( 문자열 변수, 한번에 읽어들일 최대 길이, 파일 포인터 변수 )는 파일에서 최대 길이 이하로 한 줄(\n) 단위로 읽어서 문자열 변수에 저장


stdin 
  키보드 등의 표준 입력


strstr 
  string string의 줄임말로, strstr( 대상문자열, 검색할 문자열 )은 대상문자열에서 검색할 문자열를 찾은 위치의 주소값을 반환하고 못 찾으면 NULL을 반환

예를 들어 변수공간 a  Hello, World! 와 변수공간b  World 에 대해서 strstr( a, b )을 실행하면 변수공간a에서 'W'가 저장된 위치의 주소값을 반환



%s 
  문자열  형식으로 입출력
